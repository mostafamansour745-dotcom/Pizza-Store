# Pizza Order Application

A desktop-based graphical user interface (GUI) application built using C# and Windows Forms. The project focuses on form layouts, state management, and real-time user preference processing.

---

## Key Features

* **Dynamic Size Selection:** Allows users to choose pizza sizes (Small, Medium, Large) with real-time base price adjustment.
* **Toppings Configuration:** Multiple checkbox selections for ingredients (Extra Cheese, Mushrooms, Tomatoes, Onions, Olives, Green Peppers).
* **Crust Customization:** Radio button controls to select the preferred crust type (Thin Crust, Thick Crust).
* **Dining Options:** Simple selection toggles for "Eat In" or "Take Out" orders.
* **Live Order Summary:** A reactive summary panel that displays selected options, calculated toppings, final order details, and the total price dynamically.
* **Form Controls:** Features fully functional "Order Now" and "Reset Form" capabilities to flush or process data instantly.

---

## Technical Stack and Concepts

* **C# Windows Forms:** Developed using standard desktop UI controls (RadioButtons, CheckBoxes, GroupBoxes, Buttons, and Labels).
* **State Management:** Tracking continuous user selections and updating variable pricing instantly using event-driven programming (Event Handlers).
* **UI/UX Design:** Clean, simple, and thematic visual design suitable for client-side point of sale (POS) utilities.
